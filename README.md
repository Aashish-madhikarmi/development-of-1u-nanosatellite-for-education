The **1U CubeSat** is a miniature satellite designed for academic purposes. It integrates:
- On-Board Computer (OBC): Raspberry Pi Zero W  
- Power System: Custom 3.7V LiPo with boost and regulation circuits  
- Sensors:** MPU6050 (IMU), temperature sensors, current sensors  
- Communication: SDR-based radio with SMA connector  
- Actuation: Magnetorquers for attitude control  
- Structure: 3D-printed modular frame
